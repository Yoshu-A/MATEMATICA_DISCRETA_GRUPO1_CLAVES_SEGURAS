# Secure Password Combinatorial Optimization Project

## Description

This project was developed for the Discrete Mathematics course in Software Engineering. It implements a Python prototype for secure password generation, validation and combinatorial analysis.

## Discrete Mathematics Topics

- Fundamental counting principle
- Permutations
- Combinations
- Inclusion-exclusion principle
- Algorithmic complexity analysis

- pandas
